前言

JavaScript 作为一门动态语言，有一些独特而强大的特性。深入理解闭包、原型链和异步编程，是从会用 JavaScript 到精通 JavaScript 的必经之路。本文将深入剖析这些核心概念。

一、闭包（Closure）

什么是闭包？

闭包是指函数能够记住并访问其词法作用域，即使函数在其词法作用域之外执行。

总结

闭包要点

  • ✅ 函数能够访问其词法作用域
  • ✅ 用于数据私有化、函数柯里化、模块模式
  • ⚠️ 注意内存泄漏问题

原型链要点

  • ✅ JavaScript 使用原型链实现继承
  • ✅ __proto__ 指向原型对象
  • ✅ 原型方法比实例方法更高效

异步编程要点

  • ✅ 回调 → Promise → async/await
  • ✅ 理解事件循环和宏任务/微任务
  • ✅ 合理使用并行和串行执行
